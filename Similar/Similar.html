Description

Similar is a web-based application that generates a new recommended playlist based on an existing playlist provided by the user. The app supports both Spotify and Apple Music as sources, allowing you to choose your preferred music platform.

To generate recommendations, Similar functions as a Gemini AI wrapper. Once a playlist is submitted, the application sends it through the Gemini model to analyze the tracklist and identify 30 new songs that match the style, genre, and mood of the original. The recommended songs are then displayed directly on the web interface for easy viewing and access.

How-To

Using Similar is straightforward:

  1. Select your preferred music platform: Spotify or Apple Music.
  2. Paste the URL of a public playlist from the chosen platform into the search box.
  3. Click the "Generate" button.
Within moments, Similar will display 30 recommended tracks on the screen, curated to match the feel of your original playlist.

Documentation

Prerequisites

Similar requires the following before it can run:

  • Node.js 16 or higher - Required for running the backend API calls and handling Gemini requests.
  • Spotify Web API key (for Spotify support) - Stored in a .env file for secure use.
  • Apple Music API key (for Apple Music support) - Also stored in the .env file.
  • Gemini API key - Used to access the Gemini AI model for playlist recommendations.

Configuring Environment Variables

All API keys must be placed in the .env file located in the project root. Example: 

    SPOTIFY_API_KEY=your_spotify_key_here
    APPLE_MUSIC_API_KEY=your_apple_music_key_here
    GEMINI_API_KEY=your_gemini_key_here

Ensure these keys are kept private and never exposed in public repositories.

Frontend

The frontend is built using HTML, CSS, and JavaScript, with a responsive layout for desktop and mobile devices. It includes input fields for playlist URLs, a platform selector, and a results area to display recommended songs.

Backend

Gemini Integration

The backend acts as a Gemini AI wrapper, sending playlist data for analysis. Gemini processes the submitted playlist, evaluates track characteristics, and returns 30 curated recommendations.

APIs

Depending on the chosen platform, the backend interacts with either:

  • Spotify Web API
  • Apple Music API

Display of Results

Once the 30 recommended tracks are returned from Gemini, they are displayed dynamically on the results page. Each track listing includes the song title, artist, and an optional direct link to the song on the selected music platform.
