Description

Battleship Bot is a C# project developed as part of a group assignment where the goal was to create an automated Battleship player capable of competing against other bots. The bot uses strategic algorithms to place ships, target opponents, and optimize its chances of winning the game.

How-To

To use Battleship Bot:

  1. Compile the C# project using your preferred IDE (e.g., Visual Studio).
  2. Run the bot executable to start a match against another bot.
  3. The bot will automatically place its ships and make moves without user input.
  4. Observe the match progress via the console or UI output (depending on implementation).

Documentation

Prerequisites

Battleship Bot requires:

  • .NET Framework or .NET Core SDK installed - Compatible with the C# version used in the project.
  • An IDE or compiler capable of building C# projects (Visual Studio recommended).

Project Structure

The project consists of multiple classes handling different aspects of the bot:

  • Ship - Manages ship placement and tracking.
  • Board - Represents the game board state.
  • BotLogic - Contains the targeting and decision-making algorithms.
  • GameController - Coordinates game flow and bot interactions.

Bot Strategy

The bot employs a combination of probability-based targeting and pattern recognition to efficiently find and sink opponent ships. It adapts its strategy based on hit/miss feedback to improve accuracy over time.
