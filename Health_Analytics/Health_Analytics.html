Description

Michigan Analytics is a web application designed to visualize the relationship between household income and health disparities across Michigan counties. The interactive interface features a map of Michigan split into individual counties, allowing users to either click on a county or search for it by name.

Once a county is selected, two line graphs are displayed:

  • Household income trends over the years for the selected county.
  • The county's health ranking compared to the rest of the state over time.
All health and income data is sourced from the Michigan Census Bureau and stored locally in CSV files for efficient access and analysis.

Documentation

Prerequisites

Michigan Analytics requires the following:

  • A modern web browser - Chrome, Edge, or Firefox recommended for best performance.
  • Node.js 16 or higher - For backend data handling (if server-side processing is used).
  • CSV data files - Containing household income and health ranking data sourced from the Michigan Census Bureau.

Data Configuration

The application reads data from two primary CSV datasets:

  • income.csv - Contains annual household income figures for each county over multiple years.
  • health.csv - Contains annual health ranking data for each county compared to the rest of Michigan.
These files should be placed in the /data directory of the project. The file format must remain consistent for the application to parse them correctly.

Frontend

The frontend is built with HTML, CSS, and JavaScript, featuring:

  • An interactive map of Michigan with clickable counties.
  • A search bar for quickly locating specific counties.
  • Two dynamic line graphs rendered using a charting library (e.g., Chart.js or D3.js) to visualize income and health data.

Backend

Data Handling

The backend reads and processes CSV data from the /data directory. Data is filtered by county selection, then sent to the frontend for graph rendering.

Map Integration

The county map is generated using a GeoJSON or SVG-based representation of Michigan. Click events and search queries trigger updates to the displayed graphs.

Data Source

All datasets originate from the Michigan Census Bureau and are periodically updated to ensure accurate trend analysis.
